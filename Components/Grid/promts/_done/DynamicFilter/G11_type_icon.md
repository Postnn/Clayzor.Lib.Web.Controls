# G11 — Тип 9 (Пиктограмма): 3-колоночный подзапрос

Создать `ClayIconColumnType : ColumnTypeDescriptor` (Тип 9). Формат = SQL (1 — значение,
2 — tooltip, 3 — href иконки, может быть относительным).
- Подзапрос — ОДИН раз (кэш `value→(tooltip,href)`), не на строку, ТОЛЬКО через
  `DynamicSql.QueryTriplesAsync(Db, formatSql)` (Lib.Entities, G1b). SQL из Controls не выполнять.
- Вывод: `<img>`/иконка по href с tooltip; относительный href резолвить как в `ClayGridLinkResolver` (G5).

Проверка (TG7 + ручная): колонка рендерит иконку с подсказкой; подзапрос выполнен один раз.
