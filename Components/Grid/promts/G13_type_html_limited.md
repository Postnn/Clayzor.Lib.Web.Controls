# G13 — Тип 8 (HTML), Тип 12 (строка с ограничением), санитизация; Тип 4 (Ссылка)

Создать/дополнить дескрипторы:
1. Тип 8 (`ClayHtmlColumnType`): выводить значение как HTML, но САНИТИЗИРОВАТЬ — вырезать/
   экранировать теги `<script>` и заблокировать выполнение скриптов (событийные атрибуты onXxx,
   `javascript:`). Вынеси санитайзер в чистую функцию `ClayHtmlSanitizer.Sanitize(string html)`.
2. Тип 4 (Ссылка): значение — html-гиперссылка; применить ту же санитизацию.
3. Тип 12 (`ClayLimitedTextColumnType`): длина из Формат (число). Если текст длиннее — обрезать
   и добавить «…»; полный текст — в tooltip.

Не делай: не выводи сырой HTML без санитизации.

Проверка (TG8):
- `Sanitize("<b>ok</b><script>alert(1)</script>")` не содержит исполняемого script, `<b>ok</b>` сохранён;
- Тип 12 при длине 5 и тексте "abcdefgh" → "abcde…" и полный текст в tooltip.
